# Medicine Order System
A web-based Database Management System (DBMS) designed to streamline the process of ordering medicines, managing inventory, and tracking customer details.

## Features
-> Medicine Catalog: View available medicines with pricing and details.
-> Order Processing: Seamlessly place and track medicine orders.
-> Inventory Management: Automated updates to stock levels after each purchase.
-> Secure Database: Relational database design using MySQL for data integrity.

## Tech Stack
* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: XAMPP (Apache & MySQL)

## How to Run
1. Start XAMPP: Open XAMPP Control Panel and start Apache and MySQL.
2. Setup Database:
   - Go to 'localhost/phpmyadmin'.
   - Create a new database named 'medicine_db'.
   - Click the Import tab and upload the '.sql' file (from sql folder) included in this repository.
3. Project Setup:
   - Move the project folder to 'C:/xampp/htdocs/'.
4. Access in Browser:
   - Open your browser and type: 'localhost/MediBuddy'
